# Entrega - Actividad 1

## Parte 1: Diseno y Marca del Sitio Web Comercial

### Idea de negocio
VerdeVivo Market es un e-commerce internacional de productos naturales para bienestar diario: infusiones botanicas, skincare vegetal, suplementos naturales y aceites esenciales.

### Propuesta de valor
Productos de formulas limpias y origen trazable, con experiencia de compra adaptada por idioma y moneda para mercados internacionales.

## Parte 2: Servicio al Cliente para E-commerce

### Problemas y necesidades
- Demoras en envios internacionales.
- Dudas sobre ingredientes, uso y devoluciones.
- Soporte bilingue para usuarios globales.
- Confianza y seguimiento de compra.

### Gestion de comentarios
- Positivos: agradecimiento + fidelizacion.
- Negativos: respuesta empatica publica + resolucion privada + cierre transparente.
- Metricas: CSAT, NPS, tiempo de respuesta y tasa de resolucion.

Documento asociado: `customer-service.html`

## Parte 3: Politica de Privacidad

Se redacto politica con:
- Responsable del tratamiento.
- Tipos de datos y finalidad.
- Base legal.
- Derechos del usuario.
- Conservacion y transferencias internacionales.

Documento asociado: `privacy.html`

## Parte 4: Sitio Web (WordPress o similar)

Se desarrollo un sitio funcional y responsive con:
- Home de tienda natural: `index.html`
- Cambio de idioma ES/EN: `script.js`
- Diseno visual premium: `styles.css`
- Panel admin demostrativo: `admin.html`

### Evidencia solicitada
La actividad exige captura del administrador del sitio:
- Abrir `admin.html`

- Adjuntarlo a la entrega
